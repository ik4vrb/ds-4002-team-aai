## Data
### Main Data Set
For this project there was one main data set containg information about the top 25 books over the last 125 years. This data set contained the following:

|    Column     |  Description  |
| ------------- | ------------- |
| Response_id    | Book ID for each individual book |
| Story_name     | The novel title |
| Author         | The name of the author for each book |
| Year_published | The year the book was published |
| Genre          | A general genre the book belong to |
| Response       | Around the first 250 word of each book |
| num_words      | The exact number of words that the hook has |

### List of Words
For the inital EDA, a list of around 1000 positive and negative words is needed. This data set contained the following:

|    Column     |  Description  |
| ------------- | ------------- |
| word          | A single word |
| category      | Whether the word is positive or negative |
